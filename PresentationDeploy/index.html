What is a Canvas?

A canvas is an HTML5 element designed to create a graphic using JavaScript
Canvas graphics are raster and typically 2D
There are various developments of the 3D version of the API, but at the moment there is no single specification

Canvas using for:

  • Data visualization
  • Diagrams drawing
  • Animation
  • Low-level game graphic

Canvas vs SVG

Canvas
  • Raster graphic
  • High performance
  • Image and video support
  • Content export
SVG
  • Vector graphic
  • Low performance
  • Available for DOM
  • CSS-animations
SVG - for icons and other user interface, canvas - for complex graphics and animation

The element

  • Pair tag
  • Only two attributes: width ahd height

Sizes and grid

Width and height can be set in attributes or css style
Important! Any resizing will erase everything drawn on the canvas
The origin of the grid is in the upper left corner
The coordinate system is the same as the standard browser
One grid step is usually equal to one pixel

Context


            const canvas = document.body.getElementById('.my-canvas');
            const ctx = canvas.getContext('2d');
We need a getContext method to access the contents of the canvas. The value is set to 2D

              ctx.fillRect(5, 5, 50, 50);

Shapes. Paths


              ctx.fillStyle = 'rgb(50, 200, 200)'
              ctx.beginPath();
              ctx.moveTo(45, 55);
              ctx.lineTo(155, 25);
              ctx.lineTo(75, 105);
              ctx.fill();
              
              ctx.strokeStyle = 'rgb(200, 100, 200)'
              ctx.beginPath();
              ctx.arc(150, 150, 50, 0, Math.PI * 2, true);
              ctx.closePath();
              ctx.stroke();
              
              ctx.fillStyle = 'rgb(200, 70, 00)'
              ctx.beginPath();
              ctx.arc(270, 270, 50, 0, Math.PI, false);
              ctx.lineTo(270, 100);
              ctx.closePath();
              ctx.fill();

Animation

You can animate any drawn object, for example, a simple circle 

              function draw(x, y) {
                ctx.fillStyle = 'rgb(200, 100, 200)'
                ctx.beginPath();
                ctx.arc(x, y, 5, 0, Math.PI * 2, true);
                ctx.closePath();
                ctx.fill();
              }

Animation

To create an animation, the canvas is cleared, after which the updated content is drawn on it. It repeats every frame 

              let a = 200; b = 5, dx = 2, dy = 2;
              
              function animate() {
                ctx.clearRect(0, 0, 400, 400);
                if ((a + dx) >= 395 || (a + dx) <= 5) {
                  dx = -dx;
                }
                if ((b + dy) > 395 || (b + dy) < 5) {
                  dy = - dy;
                }
                a += dx;
                b += dy;
                draw(a, b);
                window.requestAnimationFrame(animate);
              }
